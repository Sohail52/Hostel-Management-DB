# Hostel Management System (PHP & MySQL)

A simple web-based Hostel Management System built with **PHP** and **MySQL** for managing hostel rooms, students, and user authentication.

## Features

- **Admin Panel**: Manage rooms, students, and users.
- **User Panel**: View and book rooms.
- **Authentication**: Admin and User logins.

## Installation

### 1. Download & Extract Files
- Download the **Hostel Management System** zip file.
- Extract it to a folder.

### 2. Move Files to XAMPP Directory
- Move the extracted files to the `htdocs` directory in your XAMPP installation (e.g., `C:\xampp\htdocs\hostel`).

### 3. Set Up Database
- Open **phpMyAdmin** (`http://localhost/phpmyadmin`).
- Create a database called `hostel`.
- Import the `hostel.sql` file from the project folder.

### 4. Access the System
- Open your browser and visit `http://localhost/hostel/`.
